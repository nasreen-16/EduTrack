# 🎓 EduTrack

A simple and efficient web application designed to help teachers manage and track student progress in one place.

EduTrack simplifies student management by organizing attendance, academic performance, achievements, and certifications digitally — reducing the need for paper records and spreadsheets.

---

## ✨ Features

* 🔐 Teacher Login & Registration
* 👨‍🎓 Student Record Management
* 📅 Daily Attendance Tracking
* 📊 Marks & Performance Management
* 🏆 Achievement Recording
* 📜 Certification Management
* 📈 Dashboard Overview
* 🔍 Quick Student Search

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/edutrack.git
```

### Move into the project folder

```bash
cd edutrack
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open in your browser

```text
http://127.0.0.1:5000
```
